This directory contains example test files for SAT solvers.
The input files are in Dimacs format.

For each input file there is a  corresponding _solution file containing
a satisfying valuation. 

The input files are:

  sudoku_mini.txt - The 4*4 Sudoku from Lecture 3
  sudoku_hard.txt - The 9*9 Sudoku from Lecture 3 ('very hard' level)
  sudoku_easy.txt - Another 9*9 Sudoku ('very easy' level)
